# API Control Obras

Backend project focused on construction work control, cost tracking, and progress management.

## Objective
Create a practical system to register projects, construction items, costs, and progress.

## Main features
- Register projects
- Register work items
- Register costs
- Track progress
- Show summary information

## Planned technologies
- Java
- Spring Boot
- SQL / PostgreSQL
- Git & GitHub

## Project status
Initial structure in progress.

## Future improvements
- Authentication
- Reports
- Frontend integration
